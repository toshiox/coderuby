# Bem-vindo ao Repositório de Gustavo Toshio

Olá! Meu nome é Gustavo Toshio, e sou um desenvolvedor de software com 8 anos de experiência nas áreas de varejo, saúde, finanças e bancos digitais. Este repositório é onde compartilho projetos, códigos e insights acumulados ao longo da minha carreira e estudos contínuos em desenvolvimento de software e segurança da informação.
